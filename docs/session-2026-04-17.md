## 1차 작업 상세

### 1. 공통 서브헤더 (PageHeader)

**배경**: 기존에는 각 페이지가 자체적으로 제목 영역을 구성하여 높이·위치가 제각각이었음. 상단바(h-14)와 동일한 높이의 공통 영역을 두어 모든 페이지에서 일관된 구조를 가지도록 개선.

**구현**:
- `src/components/page-header.tsx` 신규 생성
- 모바일: `fixed top-14` (상단바 바로 아래 고정), 데스크탑: 일반 흐름
- `maxWidth` prop으로 페이지별 최대 너비 조절 가능
- 적용 페이지: `/dashboard`, `/teams`, `/teams/[id]`, `/settings`

**페이지별 내용**:
- 내 시프트: 제목 + 마지막 동기화 시간 + 새로고침 버튼
- 팀 목록: "팀" 제목
- 팀 상세: 팀 이름 (수정 가능) + 멤버 수 + 멤버/초대 버튼
- 설정: "설정" 제목

### 2. 내 시프트 달력 주말 색상 통일

**배경**: 팀 시프트에서는 요일 헤더(토, 일)에 배경색 오버레이가 적용되어 있었지만, 개인 달력에서는 텍스트 색상만 변경되고 배경은 동일했음.

**변경**: `calendar.tsx` 요일 헤더에 `backgroundImage: linear-gradient(overlay, overlay)` 방식으로 주말 오버레이 적용.

### 3. globals.css 색상 변수 정리

**변경 내용**:
- 모든 변수를 카테고리별로 그룹화 (배경, 텍스트, Primary, Secondary, 테두리, 입력, 상태, 이벤트, 달력, 간격)
- 각 변수에 한글 주석으로 용도 설명 추가
- 주말 배경 변수명 변경: `--weekend-sat-bg` → `--overlay-weekend-sat` (오버레이 개념 명확화)
- `--today-bg`, `--today-text` → `--overlay-today` (1차), 이후 2차에서 `--today-border`로 최종 변경

### 4. 오늘 날짜 표시 방식 변경

**1차 구현**: 숫자에 동그라미 → 셀 전체에 반투명 오버레이.
`linear-gradient(var(--overlay-today), var(--overlay-today))` 기법으로 기존 배경색 위에 반투명 레이어를 겹쳐 표현.

**2차에서 재변경**: 색상 오버레이 → 테두리 방식 (아래 참고).

### 5. 팀 페이지 플로팅 푸터 (PageFooter)

**배경**: 팀 목록 페이지에 인라인 팀 생성 폼(input + 버튼)이 즉시 노출되어 있었는데, 불필요하다고 판단.

**구현**:
- `src/components/page-footer.tsx` 신규 생성
- 하단 탭바 위에 투명 배경으로 위치하여 콘텐츠를 가리지 않음
- "팀 참여하기", "팀 생성하기" 두 개의 둥근(rounded-full) 버튼 배치
- 기존 인라인 생성 폼 제거, `prompt()` 방식으로 임시 대체 (2차에서 모달로 교체)

## 2차 작업 상세

### 6. PageHeader 테두리 제거 + 다크모드 배경

**배경**: 다크모드에서 서브헤더의 `bg-card`가 상단바와 동일한 색상이라 상단바가 2개 있는 것처럼 보임. 서브헤더는 콘텐츠 영역과 하나로 느껴져야 함.

**변경**:
- `border-b` 제거
- `bg-card` → `bg-base`로 변경 (다크모드에서 콘텐츠 배경과 동일한 색)

### 7. PageFooter 버튼 디자인 조정

**변경**:
- 버튼 세로 패딩: `py-2.5` → `py-3.5`
- 하단 탭바와의 여백: `bottom-14` → `bottom: calc(3.5rem + 1rem)` (좌우·버튼 사이 여백 16px과 동일)
- 데스크탑: `bottom: 1rem`

### 8. CreateTeamModal + 모달 외부 클릭 닫기

**신규 파일**: `src/components/create-team-modal.tsx`
- JoinTeamModal과 동일한 디자인 언어 (rounded-2xl 카드, 반투명 배경)
- 팀 이름 입력 필드 + 생성/취소 버튼

**외부 클릭 닫기**:
- `JoinTeamModal`, `CreateTeamModal` 모두 적용
- 방식: backdrop `div`에 `onClick={onClose}`, 내부 카드에 `onClick={e => e.stopPropagation()}`

**teams/page.tsx 변경**: `prompt()` 제거 → `showCreateModal` 상태로 CreateTeamModal 표시.

### 9. 팀 시프트 달력 테두리 일관성

**문제**: 일부 셀은 두꺼운 테두리, 일부는 테두리 없음. 인라인 `borderTopWidth: 2` 등 불일치.

**해결**:
- `borderCollapse: "collapse"` 적용
- 모든 데이터 셀에 일관된 `borderTop: 1px solid var(--border-light)` + `borderLeft: 1px solid var(--border-light)` 적용
- 주(week) 구분선은 `borderTop: 1px solid var(--border)` (약간 더 진한 색)
- 이름 열의 `borderRight` 유지

### 10. 오늘 날짜 테두리 강조

**변경**: 색상 오버레이 → `box-shadow: inset 0 0 0 1.5px var(--today-border)` 테두리 방식.

**globals.css**:
- `--overlay-today` 제거
- `--today-border` 추가 (라이트: `oklch(0.45 0 0)`, 다크: `oklch(0.65 0 0)`)

**적용 범위**:
- 개인 달력 (`calendar.tsx`): 셀 전체에 inset box-shadow
- 팀 시프트 (`teams/[id]/page.tsx`): 헤더 셀(요일+날짜)에만 inset box-shadow
