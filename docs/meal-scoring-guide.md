# 직원 식당 메뉴 평가 기준

매주 식단표 이미지를 받으면 이 기준표를 참고하여 **중식 / 석식 각각** 5점 만점 점수를 매긴다.

---

## 평가 항목 (총 5점)

### 1. 메인 메뉴 매력도 (2점)

| 점수 | 기준 |
|------|------|
| 2.0 | 특식/인기 메뉴 (돈까스, 불고기, 제육, 갈비, 치킨, 탕수육 등) |
| 1.5 | 대중적 메뉴 (찌개류, 볶음류, 카레, 비빔밥 등) |
| 1.0 | 무난한 메뉴 (국밥류, 조림류 등) |
| 0.5 | 호불호 강한 메뉴 (생선 조림, 특정 나물 위주 등) |

### 2. 반찬 구성 다양성 (1점)

| 점수 | 기준 |
|------|------|
| 1.0 | 5종 이상, 국/찌개 + 단백질 반찬 + 채소 반찬 조화로움 |
| 0.5 | 3~4종, 기본 구성 |
| 0.0 | 2종 이하, 단조로움 |

### 3. 영양 균형 (1점)

| 점수 | 기준 |
|------|------|
| 1.0 | 단백질(고기/생선/두부) + 채소(나물/생채/샐러드) + 탄수화물(밥/면) 3박자 |
| 0.5 | 2가지만 충족 |
| 0.0 | 탄수화물 위주 또는 한쪽으로 치우침 |

### 4. 특별 보너스 (1점)

| 점수 | 기준 |
|------|------|
| 1.0 | 특식 표시(빨간 강조) 메뉴가 있고 매력적 |
| 0.5 | 특식은 있으나 보통 수준 |
| 0.0 | 특식 없음 / 특별한 게 없음 |

---

## 점수 해석

| 총점 | 해석 |
|------|------|
| 4.5 ~ 5.0 | 무조건 가야 함 |
| 3.5 ~ 4.0 | 충분히 갈만함 |
| 2.5 ~ 3.0 | 가도 괜찮음 |
| 1.5 ~ 2.0 | 다른 곳 알아보기 |
| 0.0 ~ 1.0 | 편의점 추천 |

---

## 코멘트 작성 규칙

- 한 줄로 핵심 요약 (30자 내외)
- 메인 메뉴에 대한 한 마디 포함
- 예시:
  - `"돈까스 특식에 반찬 구성도 알찬 날"`
  - `"비빔밥 무난, 반찬이 아쉬움"`
  - `"생선조림 호불호 갈림, 패스해도 무방"`

---

## 입력 형식

매주 이미지를 전송하면 아래 플로우를 따른다:

1. 이미지에서 메뉴 데이터 파싱
2. 각 날짜/끼니별 점수 산출
3. 미리보기 테이블로 보여주기
4. 확인 후 DB에 upsert 실행

---

## 메뉴 타입 분류

| meal_type | 설명 |
|-----------|------|
| `lunch` | 중식 메뉴 전체 (국/밥/반찬 포함) |
| `dinner` | 석식 메뉴 전체 |
| `salad` | 샐러드 코너 (중식 시간에 제공) |

---

## is_featured 기준

이미지에서 **빨간색/강조색으로 표시된 메뉴** = `is_featured: true`
나머지 일반 메뉴 = `is_featured: false`
