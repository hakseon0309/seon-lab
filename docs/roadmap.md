# 로드맵

## 제품 기능

- 개인 달력 아래 특정 사용자의 시프트를 함께 보여주는 커플 기능 고도화.
- 팀 내 채팅방.
- 팀 내 근무 일정 교환 커뮤니티 개선.
- 특정 인원을 선택해 모두 근무가 없는 가장 빠른 날 찾기.
- 약속 시간대를 기준으로 가능한 날짜 찾기.
- 캘린더 공유 URL을 아이폰 카메라로 촬영해 자동 입력하는 흐름.
- 개발자 피드백 게시판.
- 개발자 패치노트 게시판.
- 게시판 생성/운영을 더 쉽게 만드는 관리자 기능.

## 안정성

- 아이폰 웹앱 또는 Safari에서 장시간 미사용 후 재접속 시 화면이 멈춰 보이는 현상 추가 점검.
- 푸시 알림, 서비스 워커, bfcache 복귀 흐름의 실제 기기 검증.

## 완료된 주요 항목

- 스타필드 하남점 트레이더스 둘째·넷째 수요일 휴무일 감지.
- 프로필 사진 기능.
- 팀 프로필 사진 기능.
- 홈 화면 추가 안내.
- 직원 식당 메뉴 점수/코멘트 제거.
- 공통 `PageHeader`, `PageFooter`, 로딩 오버레이, 달력 주말/오늘 표시 정리.
